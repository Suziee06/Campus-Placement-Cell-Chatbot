# Campus-Placement-Cell-Chatbot🤖- Nexora
The CPC Bot- Nexora, is a chatbot made by using watsonx assistant on ibm cloud. It can answer placement related FAQs, questions related to placement process, different eligibilty criterias and so much more.
# Placement Assistant Chatbot

- Check company eligibility
- Understand placement process
- Resume preparation tips
- Interview preparation guidance

## Technologies Used

- IBM Cloud
- Conversational AI

The chatbot interacts with users and provides placement-related guidance through structured conversational flows.
